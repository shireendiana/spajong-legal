# spajong-legal

Public privacy policy for the [Spajong](https://apps.apple.com/app/spajong) iOS app, hosted via GitHub Pages.

## Why a separate repo?

The main Spajong codebase is private. On the free GitHub plan, GitHub Pages requires a public source repository. This single-purpose repo exists solely to serve the privacy policy at a public HTTPS URL while keeping the app code private.
